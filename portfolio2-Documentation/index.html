Installation

  • The zip file contains all files, however you need to perform following steps to get node_modules etc.

    • Note : Make sure this all installation process is done in command prompt.

    cd portfolio-2

    npm install

  • After Completion of installation Run following command in command prompt to start React development server.

    • npm start

  • For production build run below command.

    • npm run build
